Quick Start

Installing and using Hotglue in a project can be as easy as one, two, three.

  1. Install Hotglue
  2. Include a reference on your asp.net page to your script to load. 
        <%= HotGlue.Script.Reference("app.js") %>
  3. Include a reference to any required JavaScript files your script needs. 
        //= library("jquery.js")
    //= require("subfile.js")
    var awesome = true;
