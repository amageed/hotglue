Plug-ins
Helping to make coding a little easier one at a time.

There are five types of plug-ins that can be added to expand Hotglue. They are Frameworks, Compressors, Script References, Compilers, and Templates. You will need at least one Framework to get Hotglue to work. If we don't have the exact plug-in you are looking for, jump over to contribute to find out how to add what you need.

Frameworks

Install for MVC and Web Forms: 
Install-Package HotGlue.Aspnet
Install for Nancy: 
Install-Package HotGlue.Nancy

Compressors

Install for Uglify: 
Install-Package HotGlue.Compressor.Uglify
Install for YUI: 
Install-Package HotGlue.Compressor.YUI

Hotglue uses the default <script> tag for loading referenced scripts on a page. If you want to load them differently, you can install a script reference plug-in to change how they are loaded.

Script References

Install for LABjs: 
Install-Package HotGlue.Reference.LABjs

The compilers and templates look for specific extensions to know when to process a file. You will find the required file extensions below each of the installs. If the extension has an optional name, it is denoted by [ ].

Compilers

Install for CoffeeScript: Requires file extension .coffee 
Install-Package HotGlue.Compiler.CoffeeScript
Install for TypeScript: Requires file extension .ts 
Install-Package HotGlue.Compiler.TypeScript

Templates

For any of the template plug-ins, you must reference and download the template engine source separately. This approach was taken to ensure that compatibility can be maintained across Hotglue versions, no matter what version of the template engine you have referenced locally.

Note: If you only use one template engine, you can name all of your template files with the extension .tmpl, but if you are using more than one, all template files will need to be fully qualified. (i.e. filename.tmpl-dust)

Install for Dust: Requires file extension .tmpl[-dust] 
Install-Package HotGlue.Template.Dust
Install for EJS: Requires file extension .tmpl[-ejs] 
Install-Package HotGlue.Template.EJS
Install for Handlebars: Requires file extension .tmpl[-handlebars] 
Install-Package HotGlue.Template.Handlebars
Install for JsRender: Requires file extension .tmpl[-jsrender] 
Install-Package HotGlue.Template.JsRender
Install for Mustache: Requires file extension .tmpl[-mustache] 
Install-Package HotGlue.Template.Mustache
Install for NANO: Requires file extension .tmpl[-nano] 
Install-Package HotGlue.Template.NANO
Install for Underscore: Requires file extension .tmpl[-underscore] 
Install-Package HotGlue.Template.Underscore
Install for doT: Requires file extension .tmpl[-dot] 
Install-Package HotGlue.Template.doT
Install for jQuery: Requires file extension .tmpl[-jquery] 
Install-Package HotGlue.Template.jQuery
