Library References

Whenever you reference a library file, the referenced file is loaded as is and not parsed for additional reference tags. There are multiple ways you can reference a library file. Below is a variety of ways you can reference a library file. 

    // The standard library reference path
    //= library("jquery.js");
    // or 
    //= library('jquery.js');

    // Referencing a library without quotes
    //= library jquery.js

    // Referencing a library as an element for resharper support
    /// <reference path="jquery.js" library/>

    // Referencing a library file under a folder
    //= library("libraries/jquery.js");

If you need to reference a library in another language (i.e. CoffeeScript), all you need to do is replace the comment character with the comment character in the target language. Using the first example from above. 

    // The standard library reference path in JavaScript
    //= library("jquery.js");

    // The standard library reference path in CoffeeScript
    #= library("jquery.js");
