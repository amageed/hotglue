Script References

Default

By default, Hotglue loads all references through the web standard script element. Once the files have been processed, Hotglue will replace the script reference with the required script elements. 

    // Asp.net Page
    <%= HotGlue.Script.Reference("Accounts/app.js") %>

    // Processed Page HTML

LABjs Plug-in

If you want to change the default Hotglue script references, you can install the LABjs plug-in. 

    // Asp.net Page
    <%= HotGlue.Script.Reference("Accounts/app.js") %>

    // Processed Page HTML
