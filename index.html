Hotglue Features: everything you need for your front end code!

File References Defined within the code

Have each file declare the references it needs and automatically graph the dependency tree. You no longer need to manually list out every script needed to load the page, or maintain a separate configuration file.

View documentation

Package Files Bundle and Compress

Combine and compress multiple files for fewer HTTP requests and faster loading. When you aren't releasing, Hotglue can keep all of your files separate for easier debugging and readability.

View documentation

Module Entities Remove global variables

Eliminate the use of global variables throughout your JavaScript. Every reference is a self contained module that only exposes the objects or properties needed.

View documentation

Coffee Script Compile to JavaScript

Write and code natively in coffee script and other languages, then Hotglue will translate it on the fly. Any files that haven't changed are cached for faster loading.

View documentation
