Templates

Default

By default, Hotglue does not support any template files. If you want to support a template engine, install one or more of the template plug-ins. If you only install one template engine, all of your template files extensions can be .tmpl. If you install more than one template plug-in, you will need to fully qualify all of your template file extensions.

All template plug-ins work the same way, just save your .tmpl files with the template data you need. Below is a jQuery template example for displaying a list of errors (error.tmpl). 

    
    
        {{each errors}}
            
  • ${name}: ${value}
    • 
        {{/each}}

Once you have your .tmpl files, all you will need to do to render a template file is call the template like a method. 

    var errors = [{ name: "Phone", value: "Needs to be 10 digits" }, { name: "First Name", value: "It is required" }];
    var errorTemplate = require("error.tmpl");
    var html = errorTemplate(errors);
